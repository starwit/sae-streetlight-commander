# Street Light Commander

Based on object count from Starwit Awareness Engine + Observatory, this app will control a light using Matter protocol.

## How to run

Setup environment
```bash
pyenv local 3.13
poetry install
```

```bash
poetry run python sae-streetlight-commander/LightState.py
```
